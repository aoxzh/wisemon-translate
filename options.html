Define pattern-to-replacement rules applied before translation. Enable "Regex" for pattern-based matching, or leave unchecked for exact word-boundary matching.

Site-bound Terms

Bind terms to specific domains. These rules apply only when the current page hostname matches the domain list.

AI Context Terms

These terms are injected into the translation prompt to help the LLM understand domain-specific vocabulary. Format: one term per line as "Term: definition" or "Term: definition (context)".

Each line: Term: Definition (optional context in parentheses). The definition and context help the LLM choose the right translation.
